# ✨ Features

* 🔍 **Search Agent** — Discovers reliable sources using Tavily Search
* 📖 **Reader Agent** — Extracts and summarizes information from webpages
* ✍️ **Writer Agent** — Generates detailed, publication-quality research reports
* 🧐 **Critic Agent** — Reviews, scores, and improves report quality
* 🌐 Real-time internet research
* 📊 Professional Markdown reports with structured sections
* 📑 Automatic references and citations
* 🎨 Interactive Streamlit dashboard
* 📥 Download reports instantly
* 🎯 Modular architecture for easy customization

# 🚀 Installation

## Clone Repository

```bash
git clone https://github.com/Shwetanshmishra/Multi-agent-Research-System.git

cd Multi-agent-Research-System
```

## Create Virtual Environment

```bash
python -m venv multiagent

multiagent\Scripts\activate
```

## Install Dependencies

```bash
pip install -r requirements.txt
```

## Create `.env`

```env
MISTRAL_API_KEY=your_mistral_api_key

TAVILY_API_KEY=your_tavily_api_key
```

## Run the Application

```bash
streamlit run app.py
```

or simply

```bash
run.bat
```

---

# ☁️ Deploy on Streamlit Cloud

1. Push the project to GitHub
2. Connect your repository on Streamlit Cloud
3. Add your Secrets

```toml
MISTRAL_API_KEY="your_mistral_api_key"

TAVILY_API_KEY="your_tavily_api_key"
```

4. Deploy 🚀
